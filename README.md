# 무엇이 무엇이 똑같을까

여러 문서(엑셀·PDF·HWPX)를 한 번에 업로드해 **완전히 동일한 문장**을 찾아주는 웹앱입니다.
파일 안에서의 중복과 파일 사이의 중복을 모두 검출하며, 분석이 끝나면
Upstage **Solar 3 Pro**가 종합 의견을 작성합니다.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 http://localhost:3719 접속.

## 판정 기준

- **완전 동일 문장만** 중복으로 판정합니다. 조사가 하나라도 다르면 다른 문장입니다.
- 마침표·쉼표 등 **구두점과 공백 차이는 무시**하고 비교합니다.
- **온전한 문장만 비교**합니다: 두 어절 이상이고 끝맺음(종결부호 또는 '~함/~임/~다'
  같은 종결 표현)이 있는 문장. "성실함."(한 어절), "책임감이 강한 편"(끝맺음 없음)
  같은 조각은 비교 대상이 아닙니다.

### 서식 문구 자동 제외

문서마다 반복되는 기본 서식은 중복으로 취급하지 않습니다.

- **엑셀 머리글**: 각 시트의 첫 번째 내용 행을 열 제목으로 보고 제외합니다
  (내용이 한 행뿐인 시트는 제외하지 않음).
- **온전하지 않은 문구**: 위 문장 기준을 만족하지 못하는 텍스트 — 제목·머리글·이름·
  명사구·한 단어 표현 — 는 모두 제외합니다.
- **AI 경계 검토(Solar 3 Pro)**: 규칙이 제외한 문구를 AI가 전수 재검토해,
  규칙이 모르는 끝맺음으로 끝나는 실제 문장이 있으면 되살립니다.
  판단이 애매하면 조각으로 두고, AI 호출이 실패하면 규칙 판정을 그대로
  사용합니다(분석은 중단되지 않음).
- 제외·복원 건수는 결과 화면과 다운로드 보고서에 함께 표시됩니다.

## 지원 형식과 위치 표시

| 형식 | 추출 단위 | 위치 표시 |
|------|-----------|-----------|
| xlsx / xls / csv | 모든 셀 | `시트명!B3` |
| pdf | 페이지 | `3쪽` |
| hwpx | 문단 | `문단 12` |

## 결과 다운로드

- **엑셀**: 요약 시트 + 중복 문장 상세 시트. 셀 서식(테두리·줄바꿈·행 높이 자동 계산)이
  적용되어 긴 문장도 잘리지 않고 표시됩니다.
- **PDF**: Paperlogy 서체가 내장된 A4 보고서.

## 환경 변수 (.env)

```
UPSTAGE_API_KEY=up_...   # Upstage API 키
UPSTAGE_MODEL=solar-pro3 # 기본값. 모델명이 바뀌면 여기만 수정
```

AI 호출이 실패해도 중복 검출 결과는 정상적으로 표시됩니다(의견 자리에 오류 사유 표기).

## 구조

- `app/page.js` — 화면(업로드 → 진행률 애니메이션 → 결과 표)
- `app/api/analyze/route.js` — 파일 파싱·중복 검출·AI 의견, 진행률 NDJSON 스트리밍
- `app/api/export/xlsx/route.js` — 서식 적용된 엑셀 생성 (exceljs)
- `lib/extract.js` — xlsx/pdf/hwpx 텍스트 추출
- `lib/detect.js` — 문장 분리·정규화·완전 동일 문장 검출
- `lib/solar.js` — Upstage Solar 3 Pro 호출
- `lib/pdf-export.js` — 클라이언트 PDF 생성 (jsPDF + Paperlogy 내장)
